# ppl-web

Pay Per Lead Website — built with [Next.js](https://nextjs.org).

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy on Vercel

Connect this repository to [Vercel](https://vercel.com). Use the repository root as the **Root Directory** (where `package.json` lives). No subdirectory is required unless you move the app into a monorepo folder.
